# Next Pass Recommendation

## V1B: Secure Draft Media Upload

The next engineering pass should close the principal operator dependency rather than add more decoration.

Required outcomes:

1. Define and implement an owner-and-room-scoped draft media upload contract compatible with the nested editor draft lifecycle.
2. Store or serve draft-only media without making unpublished room changes public.
3. Validate MIME, extension, maximum size, SVG policy, and metadata handling.
4. Connect upload progress and failure states to the Media drawer.
5. Prove Canvas, private preview, publish, public output, and RoomKey boundaries with uploaded media.
6. Decide and implement a real focal-point field only after renderer parity is explicit.
7. Run a hosted guided owner session focused on Images and Draft/Live comprehension.

Do not expand widget/customisation breadth before this media autonomy gap is solved.
