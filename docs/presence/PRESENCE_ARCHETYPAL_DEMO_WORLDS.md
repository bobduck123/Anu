# Presence Archetypal Demo Worlds

These are not placeholders. They are prototype worlds.

Each one should test whether the system can carry a particular kind of public identity.

The first real pilots will need bespoke content. These worlds exist so product, design, sales, and implementation can judge whether Presence can carry the right kinds of people, practices, venues, and cultural organisations before live client content arrives.

## 1. Nara Vale - The Threshold Artist

- Template: Minimal Artist Portal
- display_mode: `minimal_portal`
- Mechanism: threshold / private exhibition
- Primary verb: enter
- World statement: A threshold of projected bodies, soft architectures, and rooms that remember light.

### First 10 Seconds

A dark field. A suspended image. A name that feels like a signal. One line of copy. A single Enter action.

The visitor should not yet be browsing. They should be waiting at a doorway.

### Public Identity

Nara Vale works with projection, fabric, skin, shadow, and architectural darkness. Her practice sits between installation, performance documentation, and moving-image stillness.
