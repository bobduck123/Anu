# Presence Nodes Tiers And Pricing

Presence Nodes are positioned as boutique public presence builds attached to a scalable network object model. Pricing below is reference positioning for alpha sales and delivery scoping, not hard-coded billing logic.

## Basic Node

- Digital card/profile
- Aesthetic `profile_card` template
- Links
- Simple enquiry
- QR display
- vCard
- Basic SEO/social preview
- No advanced workflow modules
- Reference: `$199-$499` setup plus `$19-$39/month`

## Premium Node

- Premium or practitioner profile renderer
- Services
- Portfolio blocks
- Custom mood/styling fields
- Business sections
- Analytics
- Advanced enquiry capture
- Admin editing
- Directory/map/archive readiness flags
- Reference: `$699-$1,500` setup plus `$49-$99/month`

## Artist Presence

- Gallery microsite via `artist_gallery`, `gallery_portal`, or `minimal_portal`
- Optional landing portal
- Practice statement
- Curatorial statement
- Selected works
- Collections
- Artwork/project metadata
- Contact flow
- Reference: `$1,500-$3,500` setup plus `$79-$199/month`

## Organisation / Venue Node

- Organisation or venue profile renderer
- Mission/about/program sections through node sections
- Services/programs
- Venue/location fields
- Gallery/media
- Enquiry routing readiness
- Directory/map/archive readiness
- Reference: `$3,000-$8,000` setup plus `$199-$599/month`

## White-label Network

- Multi-node branded ecosystem
- Tenant-ready services
- Organisation dashboards
- Future public network layer
- White-label readiness flags and tenant-aware APIs
- Reference: `$8,000-$25,000+` setup plus `$500-$2,000+/month`

## Alpha Implementation Mapping

- `plan_type=basic`: Basic Node.
- `plan_type=premium`: Premium and practitioner nodes.
- `plan_type=artist_presence`: Artist portals and galleries.
- `plan_type=organisation_venue`: organisation and venue profiles.
- `plan_type=white_label_network`: future network entry mode.

The alpha does not implement billing, payments, marketplace workflows, or automated package provisioning. These tiers shape templates, display modes, admin editing, documentation, and future delivery boundaries.
