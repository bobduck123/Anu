# Presence Ecosystem Premium Modules

Modules are represented by `presence_business_function.function_type` and related structured tables. Alpha stores configuration and foundational records; it does not run full external workflows.

## Implemented Foundations

- `services`: services/offers editor and public service/offer rendering.
- `analytics`: privacy-conscious analytics summary.
- `credentials`: public credential/licence wallet.
- `proof_ledger`: proof items/case studies.
- `procurement_profile`: contract/procurement readiness fields.
- `professional_contract`: professional contract renderer and fields.
- `quote_requests`: public quote request flow.
- `variation_approval`: variation records.
- `invoice_support`: external invoice/manual accounting records.
- `proof_of_work_handover`: handover records.
- `nfc_lead_capture`: NFC tag records and anonymous hit tracking.
- `relationship_ledger`: named connections and timeline.
